# Estensione Chrome PrezzoLibri

## Installazione

1. Apri `chrome://extensions` in Chrome sul computer.
2. Attiva **Modalità sviluppatore** in alto a destra.
3. Premi **Carica estensione non pacchettizzata**.
4. Seleziona la cartella:
   `C:\Users\Utente\Documents\Playground\PrezzoLibri\chrome-extension`
5. Fissa l'estensione **PrezzoLibri - Raccolta marketplace** alla barra di Chrome.

## Utilizzo

1. Apri <https://beppeventuru.github.io/PrezzoLibri/>.
2. Accedi e apri il libro da valutare.
3. Premi l'icona dell'estensione e poi **Avvia raccolta**.
4. Non chiudere la scheda di PrezzoLibri finché non compare il riepilogo.

L'estensione apre in secondo piano le ricerche pubbliche, legge soltanto titolo,
prezzo, spedizione, condizioni e URL degli annunci, poi chiude le schede. Non
legge password, cookie o altre pagine della navigazione.

I marketplace supportati sono Vinted, eBay, AbeBooks, Subito, Libraccio, IBS e Amazon.
